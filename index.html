Kalkulator Luas dan Keliling Persegi

Selamat Datang di Kalkulator Luas dan Keliling Persegi

Web ini digunakan untuk menghitung luas dan keliling persegi dalam rangka menyelesaikan mini project RevoU

Rumus Luas Persegi adalah : L = s x s

Keterangan   : L = Luas persegi
               s = sisi persegi

Rumus Keliling Persegi adalah : K = s x 4

Keterangan   : K = Keliling persegi
               s = sisi persegi
